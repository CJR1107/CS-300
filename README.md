# CS 300 – Data Structures and Algorithms Portfolio

## Overview
This repository contains work from CS 300. It includes analysis of data structures and a program that sorts and prints course information in alphanumeric order.

## Reflection

The main problem I was solving in this course was how to organize and manage course data efficiently. I needed to figure out the best way to store, search, and sort the data so it could be used easily in a program.

To solve this, I used different data structures like vectors, hash tables, and binary search trees. Learning these showed me how important data structures are because they affect how fast a program runs and how much memory it uses. Choosing the right one makes a big difference.

One of the biggest challenges I had was getting the program to correctly read and sort the data from the file. I also had issues with errors in my code and making sure everything connected properly. I worked through these by testing small parts at a time and fixing errors step by step.

This project changed how I think about building programs. I now focus more on planning before coding and thinking about efficiency instead of just making something work.

It also helped me improve how I write code. I try to keep it cleaner, easier to read, and more organized so it can be updated later without breaking everything.
